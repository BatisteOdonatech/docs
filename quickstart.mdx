## Prerequisites

Before you begin, ensure you have:

- Node.js 19 or higher
- A Neon PostgreSQL database
- A Clerk account for authentication
- pnpm package manager (recommended)

## Installation

<Steps>
<Step title="Clone the repository">

```bash
git clone https://github.com/BatisteOdonatech/v0-minimalist-foosball-site.git
cd v0-minimalist-foosball-site
```

</Step>

<Step title="Install dependencies">

```bash
pnpm install
```

</Step>

<Step title="Set up environment variables">

Create a `.env.local` file in the root directory:

```bash
# Database
DATABASE_URL=your_neon_database_url

# Clerk authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
CLERK_SECRET_KEY=your_clerk_secret_key

# Admin users (comma-separated Clerk user IDs)
ADMIN_USER_IDS=user_xxx,user_yyy
```

<Tip>Get your Clerk keys from the [Clerk dashboard](https://dashboard.clerk.com)</Tip>

</Step>

<Step title="Initialize the database">

Run the SQL schema from `scripts/history/03-complete-schema.sql` in your Neon database console to create the required tables.

</Step>

<Step title="Start the development server">

```bash
pnpm dev
```

Your application will be available at `http://localhost:3000`

</Step>
</Steps>

## First steps

<AccordionGroup>
  <Accordion icon="user-plus" title="Create your first player">
    Navigate to the log game page and enter player names. Players are automatically created when you log your first game.
  </Accordion>
  
  <Accordion icon="gamepad" title="Log your first game">
    1. Go to `/log-game`
    2. Select game type (1v1, 2v1, or 2v2)
    3. Enter player names for each team
    4. Enter the final score (winning team must have 10 points)
    5. Submit the game
    
    ELO ratings will be calculated automatically!
  </Accordion>
  
  <Accordion icon="shield" title="Set up admin access">
    1. Sign in with Clerk
    2. Copy your Clerk user ID from the Clerk dashboard
    3. Add it to the `ADMIN_USER_IDS` environment variable
    4. Restart the server
    5. Access the admin panel at `/admin`
  </Accordion>
</AccordionGroup>

## Deployment

### Deploy to Vercel

<Steps>
<Step title="Connect your repository">
Push your code to GitHub and connect it to Vercel.
</Step>

<Step title="Configure environment variables">
Add all environment variables from `.env.local` to your Vercel project settings.
</Step>

<Step title="Deploy">
Vercel will automatically build and deploy your application.
</Step>
</Steps>
