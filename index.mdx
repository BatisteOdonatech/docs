## Welcome to foosball tracker

A Next.js application for tracking foosball games with ELO ratings, player statistics, and comprehensive analytics. Built with modern web technologies and designed for competitive foosball environments.

## Tech stack

Built with modern, production-ready technologies:

- **Framework**: Next.js 15 with App Router
- **Database**: Neon PostgreSQL with serverless driver
- **Authentication**: Clerk for user management
- **UI**: Tailwind CSS with shadcn/ui components
- **Charts**: Recharts for data visualization
- **Deployment**: Vercel
