## Overview

The leaderboard displays all active players ranked by their current ELO rating. It provides a quick overview of player performance with statistics and dynamic achievement tags.

## Ranking system

Players are ranked by their current ELO rating in descending order:

- **Highest ELO**: Top of the leaderboard
- **Lowest ELO**: Bottom of the leaderboard
- **Starting ELO**: 1200 (default)

## Player information

Each leaderboard entry shows:

### Core stats

- **Name**: Player identifier
- **Current ELO**: Latest rating after all games
- **Games played**: Total number of matches
- **Wins**: Number of victories
- **Losses**: Number of defeats
- **Win rate**: Percentage of games won

## Tag calculation

Tags are computed dynamically based on:

- Recent game results (last 3 games for RISING)
- Latest ELO change (for BUMP)
- Overall statistics (for BALANCED, UNFAIR)
- Historical records (for RECORD, ZHERO)

Multiple tags can be displayed simultaneously if a player meets multiple criteria.

## Viewing player details

Click on any player name to view their detailed profile at `/player/[name]`:

- Complete game history
- ELO progression chart
- Head-to-head records
- Teammate statistics
- Performance trends

## Leaderboard features

### Real-time updates

The leaderboard refreshes automatically when:
- New games are logged
- ELO ratings change
- Player statistics update

### Responsive design

Optimized for all screen sizes:
- Desktop: Full table with all columns
- Mobile: Condensed view with essential stats
- Tablet: Balanced layout

### Sorting

Players are always sorted by current ELO (highest to lowest). Future versions may support custom sorting options.

## ELO distribution

Typical ELO ranges:

- **1400+**: Elite players
- **1300-1399**: Advanced players
- **1200-1299**: Intermediate players (starting range)
- **1100-1199**: Developing players
- **Below 1100**: Beginner players

<Note>
ELO ratings are relative to your player pool. A 1300 rating in one group may not equal 1300 in another.
</Note>

## Inactive players

Players can be disabled in the admin panel:
- Disabled players don't appear on the leaderboard
- Their games remain in history
- They can be re-enabled at any time

## Statistics calculation

### Win rate

```
Win Rate = (Wins / Games Played) √ó 100
```

Displayed as a percentage with one decimal place.

### Games played

Total count of all games where the player participated, regardless of team size or game type.

### Current ELO

The player's most recent ELO rating from their latest game. If a player has no games, they display the starting ELO (1200).
